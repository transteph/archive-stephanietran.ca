HTML Calendar

#TrendingCalendar - a website that provides hashtags for both formal and informal celebratory holidays, allowing users to plan their social media posts ahead as well as aquire inspiration.

Skills Used: HTML, CSS, jQuery, Javascript


View Project

Discovery

Our first assignment required us to create an HTML calendar around a theme of our choice. Wanting to create something purposeful for others, I decided to create a calendar that would helpful for social media content creators, specifically for those working in the non-profit sector. As one of the contributors for Amnesty International Toronto Organization's Digital Media Team at the time, I found it sometimes difficult to create social media content without being redundant. This, combined with A) My great appreciation for social media web applications that allow for queuing of social media posts, and B) My motivation to drive followers and user engagement, I thought a hashtag calendar would be perfect.

Composition

The finished calendar is mobile responsive with 3 media queries that allow for optimal display across different device sizes. On both mobile and desktop devices, the calendar is responsive to the user's touch, which initiate CSS transformations and transitions that highlight the hashtags for that day.
